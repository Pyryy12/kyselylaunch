# Ohjelmistokehittäjänä toimiminen -opintokokonaisuus

Tämä materiaali tullaan opiskelemaan lukujärjestyksestä poikkeavassa järjestyksessä. Ensin rajapinnat, sen jälkeen tietokannat ja viimeisenä tietokantojen hyödyntäminen.
Kurssilla tullaan hyödyntämään Helsingin yliopiston Full Stack Open -verkkokurssia. Opintojaksolla on 3 teemaa, mutta on suunniteltu toteutettavaksi 4 jakson ajan + 1 jakso projektikurssille. Tästä johtuen kurssien sisällöt eivät välttämättä vaihdu samaan aikaan kuin jaksot vaihtuvat.

Jokaisen teeman materiaalit sekä tehtävät löytyvät omasta kansioistaan. Kaikki tehtävät palautetaan joko palautuskansioon tähän repositoryyn tai omaan repositoryynsa/branchiin nettiin julkaistavasta tehtävästä ollessa kyse.
